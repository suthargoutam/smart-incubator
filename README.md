# 🐣 Smart Incubator Project

## 📌 Description
This is an IoT-based Smart Incubator using ESP32.
It automatically controls temperature and humidity.

## 🔧 Features
- WiFi connectivity
- GSM fallback system
- Temperature monitoring
- Humidity control
- Automatic egg turning (if you have)

## 🧰 Components Used
- ESP32
- DHT11 / DHT22 Sensor
- Relay Module
- Heater / Bulb
- GSM Module
